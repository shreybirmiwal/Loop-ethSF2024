## Installation


**React Web App**
```
cd frontend
cd react-web-App
npm install --legacy-peer-deps
npm run start
```

**Smart Contracts**
To deploy on thirdweb:
```
cd crypto
cd thirdweb-deployment
npx thirdweb@latest deploy
```
**To see contract:**
```
cd crypto
cd Contracts
```

**Flask Backend**
```
cd backend
py flask_server.py
```
